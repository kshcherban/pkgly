
## What is Pkgly?
An open source artifact manager. Written in Rust back end and an Vue front end to create a fast and modern experience

## Quickstart
For the fastest setup paths, see the [Quickstart](./quickstart.md) guide. It covers both the local `docker compose` flow and the Kubernetes Helm chart flow.

## What is an artifact manager?
To first define an Artifact Manager we need to define an artifact. An artifact is a package created as a product in development. They will be usually called by other projects as libraries

Artifact Managers handle the artifacts produced allowing for retrieval and uploading. 

## Why pkgly?

pkgly is a free and open source artifact manager. Common ones such as Jrog cost money. pkgly also is simple for common developers not just for big companies. pkgly is also written using modern technology such as Vue and Rust giving it a modern feel. Compared to the clunky nature of Nexus.
